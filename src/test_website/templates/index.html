Session Security Testing Environment

Welcome to the Session Security Demo

This is a testing environment for exploring session management security features and vulnerabilities.

Use this site to generate different types of session tokens and simulate various security scenarios.

Login Vulnerable Demo

Available Security Levels

When logging in, you can choose from three security levels:

High
  • Strong algorithm (HS256)
  • Short expiration time (1 hour)
  • HttpOnly flag enabled
  • Secure flag enabled
  • SameSite=Strict
Medium
  • Strong algorithm (HS256)
  • Longer expiration time (24 hours)
  • HttpOnly flag enabled
  • Secure flag disabled
  • SameSite=Lax
Low
  • Strong algorithm but weak secret key (HS256 with "weak_secret")
  • Very long expiration time (7 days)
  • HttpOnly flag disabled
  • Secure flag disabled
  • SameSite not set

Project Purpose

This website is part of a computer security class project for testing a session management analysis tool. It deliberately includes vulnerabilities for educational purposes.

Warning: Do not use this code in a production environment.
